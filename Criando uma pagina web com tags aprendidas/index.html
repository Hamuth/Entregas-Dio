Tags de listagem

voltar ao início

As tags do tipo "lista" podem ser nomeadas em suas seguintes funcões:

  1. A tag <ol> é utilizada principalmente como uma tag ordenada, podendo ser em números ou letras para decidir as posições dos itens listados.
  • A tag <ul> é utilizada principalmente como uma tag não ordenada, sendo utilizado um pequeno circulo para decidir as posições dos itens listados.

E para listar os itens tanto em uma lista ordenada e não ordenada, utilizamos a tag <li>.

Tags sub e sup

voltar ao início

A tag <sub> é utilizada para definir um texto subscrito, um tipo de texto que aparece meio caractere abaixo da linha normal ou até mesmo em uma fonte menor. Ele pode ser utilizado também para representar fórmulas químicas, como CO2.

A tag <sup> é utilizada para definir um texto sobrescrito, um tipo de texto que aparece meio caractere acima da linha normal e também pode ser renderizadp em uma fonte menor. Ele é utilizado principalmente como notas de rodapé, como HTTP[3].
